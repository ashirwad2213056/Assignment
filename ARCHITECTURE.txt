═══════════════════════════════════════════════════════════════════════════════
                              DATA FLOW DIAGRAM
═══════════════════════════════════════════════════════════════════════════════

1. USER INTERACTION → FRONTEND
   User clicks button / submits form
   ↓
2. FRONTEND → API SERVICE
   React component calls api.js function
   ↓
3. API SERVICE → BACKEND
   HTTP request sent to http://localhost:5000/api/endpoint
   ↓
4. BACKEND MIDDLEWARE
   CORS check → JSON parsing → Route matching
   ↓
5. ROUTE HANDLER
   Process request → Validate data
   ↓
6. MONGOOSE → MONGODB
   Query/Create/Update/Delete operation
   ↓
7. MONGODB → MONGOOSE
   Return data
   ↓
8. BACKEND → FRONTEND
   JSON response sent back
   ↓
9. FRONTEND UPDATES
   React state updated → UI re-renders
